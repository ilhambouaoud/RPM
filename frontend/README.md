# RPM Dashboard – CNESTEN

## Description

RPM Dashboard est une application web développée pour le centre CNESTEN.
Elle permet la gestion et la supervision des portiques de détection de radiation (Radiation Portal Monitor).

Cette application est développée avec :
- Vue 3
- TypeScript
- Tailwind CSS
- Shadcn UI
- Vite
